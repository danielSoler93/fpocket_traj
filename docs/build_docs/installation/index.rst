Installation
====================

From Conda (recommended)
--------------------------

::

 conda create -n fpocket_env -c NostrumBioDiscovery -c conda-forge --yes

 conda activate fpocket_env

 conda install -c NostrumBioDiscovery -c conda-forge fpocket_traj
